如何理解贝叶斯定理

2019-12-30 6 min read # 统计学

贝叶斯定理

由条件概率得：

P(a∩b)=P(b)×P(a∣b)=p(a)×P(b∣a)P(a \cap b) = P(b) \times P(a \mid b) = p(a) \times P(b \mid a) P(a∩b)=P(b)×P(a∣b)=p(a)×P(b∣a)

将P(a∣b)P(a \mid b)P(a∣b)移到等式右边，推导出贝叶斯定理：

P(a∣b) =P(a)×P(b∣a)P(b)P(a \mid b) \ = \frac{P(a) \times P(b \mid a)}{P(b)} P(a∣b) =P(b)P(a)×P(b∣a)​

其中 P(b)=P(b∣a)P(a)+P(b∣ac)P(ac)P(b) = P(b \mid a) P(a) + P(b \mid a ^c) P(a^c)P(b)=P(b∣a)P(a)+P(b∣ac)P(ac)

P(a∣b) =P(a)×P(b∣a)P(b∣ac)P(a)+P(b∣ac)P(ac)P(a \mid b) \ = \frac{P(a) \times P(b \mid a)}{P(b \mid a ^c) P(a) + P(b \mid a ^c) P(a^c)} P(a∣b) =P(b∣ac)P(a)+P(b∣ac)P(ac)P(a)×P(b∣a)​

例题

假设我们有一个袋子，袋子中有5枚均匀的硬币，10枚不均匀的硬币。均匀的硬币得到正面和反面的机会为50%：50%；而非均匀的硬币，假设得到正面的机会为80%，得到反面的机会则是20%。从袋中随机取出一枚硬币，然后把这枚硬币投掷6次。

我们想要知道的是：

问题1、投掷6次，得到4次正面向上的概率是多少？

问题2、在给出投掷6次，得到4次正面的条件下，取出的是一枚均匀硬币的概率是多少？

解决问题1：取出一枚投掷6次，4次正面的概率是多少？

分析：取出一枚枚硬币，得到4次正面的概率应该为 ：

取出一枚均匀硬币，并得到4次正面的概率 + 取出一枚非均匀硬币，并得到4次正面的概率，之和。

假设:

  • 取出均匀硬币为 P(F)，取出不均匀硬币的概率为P(U) ；

  • 投掷6次，得到4次正面的概率为P(4H/6)。

得到4次正面的概率，数学公式表达如下：

P(4H/6)=P(F∩4H/6)+P(U∩4H/6)P(4H/6) = P(F \cap 4H/6) + P(U \cap 4H/6) P(4H/6)=P(F∩4H/6)+P(U∩4H/6)

1） 假设取出的是均匀硬币

取出均匀硬币的概率是:

P(F)=5(10+5)=1/3P(F) = \frac{5}{(10+5)} = 1/3 P(F)=(10+5)5​=1/3

投掷这枚枚均匀硬币6次，得到4次正面的概率是：

P(4H/6∣F)=(64)(0.5)4(1−0.5)2P(4H/6 \mid F) = \binom{6}{4}(0.5)^4(1 - 0.5)^2 P(4H/6∣F)=(46​)(0.5)4(1−0.5)2

取出的是均匀硬币，且得到4次正面的概率是：

P(F∩4H/6)=P(F)×P(4H/6)=1/3×(64)(0.5)4(1−0.5)2=7.8125%P(F \cap 4H/6) = P(F) \times P(4H/6) = 1/3 \times \binom{6}{4}(0.5)^4(1 - 0.5)^2 = 7.8125\% P(F∩4H/6)=P(F)×P(4H/6)=1/3×(46​)(0.5)4(1−0.5)2=7.8125%

2）假设取出的是不均匀硬币

取出一枚不均匀(unfair)硬币的概率是：

P(U)=10(10+5)=2/3P(U) = \frac{10}{(10 + 5)} = 2/3 P(U)=(10+5)10​=2/3

投掷这枚不均匀硬币6次，得到4次正面的概率是：

P(4H/6∣U)=(64)(0.8)4(1−0.8)2P(4H/6 \mid U) = \binom{6}{4}(0.8)^4(1-0.8)^2 P(4H/6∣U)=(46​)(0.8)4(1−0.8)2

取出不均匀的硬币，且得到4次正面的概率是：

P(U∩4H/6)=P(U)×P(4H/6)=2/3×(64)(0.8)4(1−0.8)2=16.384%P(U \cap 4H/6) = P(U) \times P(4H/6) = 2/3 \times \binom{6}{4}(0.8)^4(1 - 0.8)^2 = 16.384\% P(U∩4H/6)=P(U)×P(4H/6)=2/3×(46​)(0.8)4(1−0.8)2=16.384%

最后可得4次正面向上的概率为：

P(4H/6)=P(F∩4H/6)+P(U∩4H/6)=24.1965%P(4H/6) = P(F \cap 4H/6) + P(U \cap 4H/6) =24.1965\% P(4H/6)=P(F∩4H/6)+P(U∩4H/6)=24.1965%

解决问题2：在投掷6次，得到4次正面条件下，取出的是一枚均匀硬币的概率是多少？

P(F∣4H/6)=P(4H/6∣F)P(F)P(4H/6)=7.8125%24.1965%=32.3%P(F \mid 4H/6) = \frac{P(4H/6 \mid F) P(F)}{P(4H/6)} = \frac{7.8125\%}{24.1965\%}=32.3\% P(F∣4H/6)=P(4H/6)P(4H/6∣F)P(F)​=24.1965%7.8125%​=32.3%
